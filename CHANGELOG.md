# Changelog

All notable changes to Teleprompter will be documented here.

## [1.0.0] — 2026-04-03

### Added
- Always-on-top floating window
- Screen share invisibility (macOS `NSWindowSharingNone`, Windows `WDA_EXCLUDEFROMCAPTURE`)
- CSS-based window opacity slider (works on both platforms)
- Auto-scroll with adjustable speed (0.1× – 10×)
- Reverse scroll mode
- Mirror text mode (for physical teleprompter hardware)
- Click-through / pass-through mode with keyboard shortcut reminder
- Save / load scripts via native file dialog
- AI script summarizer (Anthropic Claude API)
- Dark / Light / Green / Amber themes
- Font family, size, alignment, line spacing controls
- Floating HUD toolbar with auto-hide during playback
- HotkeyStrip: compact always-visible controls bar
- Full keyboard shortcuts: Space, ↑↓, Esc, R, M, T
- Eye-line indicator at 40% vertical position
